# 快速开始示例

这个专题是一组给新用户准备的示例文档，用来演示 `Docs Atlas` 推荐的内容组织方式。

## 你会看到什么

这个目录下包含：

- 一个 `README.md`，作为专题入口页
- 多篇按顺序编号的 Markdown 文档
- 文档之间的相对链接
- 代码块与配置示例

## 推荐阅读顺序

1. [准备你的文档目录](./01-prepare-your-docs.md)
2. [配置多个文档来源](./02-configure-multiple-sources.md)
3. [用 AI 生成可直接接入的文档](./03-ai-docs-workflow.md)

## 这组示例的目的

这不是业务文档，而是一个最小可运行示例。

你可以直接参考这个目录结构，快速开始整理自己的：

- 项目设计文档
- 教程文档
- 接入说明
- AI 生成文档
