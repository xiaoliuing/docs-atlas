下载时建议直接以 `Releases` 页面中的最新桌面端版本为准。

## 对当前产品的正确理解

可以把 Docs Atlas 理解成两层：

- 桌面端：长期使用的本地文档入口，负责聚合、阅读、搜索、编辑和保存
- Web 端：只读文档站，负责预览、构建和发布

如果你只是想把本机文档先跑起来，优先关注桌面端。
