## 建议的理解方式

你可以把 Docs Atlas 理解成两层：

- 桌面端负责长期使用、本地知识库管理和文档聚合
- Web 端负责预览、发布和只读分享

如果你只是想开始用这个项目，优先从桌面端开始。
